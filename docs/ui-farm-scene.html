HUD Bar
💰 Currency Pills
Coins and Farm Tokens shown as compact pills. Farm Tokens (🌾 FT) are earned on every harvest — amount depends on crop rarity (Lv.1 crops = 1 FT, premium crops = 10–15 FT). The season badge affects which crops are available and their yield multipliers. Tapping the season badge opens the seasonal calendar.
🗺️ Zone Tabs
Scrollable horizontal tab strip. Active zone highlighted emerald. Locked zones show level requirement greyed out. A newly-unlocked zone gets a ✨ New! gold highlight until first visited. Tapping a zone smoothly pans the grid camera to that zone.
Overlay Badge System
📍 Four Badge Types per Tile
💧 Water (blue, top-right) — crop was watered this cycle; yields max output
👷 Worker (gold, top-right) — a Field Hand is currently tending this plot
7m Wilt (red, top-right) — countdown to total yield loss; blinks when <5 min
47/100 Fill (gold, top-left) — barn or building capacity indicator
Wilt Warning System
⚠️ Persistent Wilt Banner
Appears above the toolbar whenever any crop enters the wilt window. Shows crop name and exact minutes remaining. Direct Harvest button — tapping also zooms the grid to the wilting tile. If multiple crops are wilting, the banner shows "3 crops wilting!" and tapping cycles through them. Banner disappears once all wilting crops are cleared.
Slide-Up Tooltip Design
Tile Tooltip — Four Stat Grid
Tapping any non-path tile slides up a context sheet from the bottom. The four stats shown are always: Status · Yield · Grow time · Wilt timer. Tags (💧 Watered, 🌿 Fertilised, 👷 Worker) show at a glance. The primary action button (gold, full-width) is always the single most important thing to do: Harvest, Plant, Add to Queue, or Collect.
Processing Building Tooltip
Shows active recipe with progress bar, queue depth, inputs consumed, expected output, and time remaining. Set Priority button lets the player reorder queued jobs. Kitchen Hand badge shown when assigned (unlocks at Lv.25).

Worker Dot
👷 Animated Worker Patrol
A small gold avatar walks a looping patrol path across the grid, representing the active Field Hand. Multiple workers show multiple dots with different paths and animation offsets. Tapping the dot opens the Workers panel pre-scrolled to that worker. The dot never overlaps building tiles.
