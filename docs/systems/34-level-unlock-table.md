## 📊 Unlock Density Chart

| Level Range | Unlocks per 10 Levels |
|-------------|----------------------|
| 1–10 | ~18 (tutorial heavy) |
| 11–20 | ~12 |
| 21–30 | ~10 |
| 31–40 | ~8 |
| 41–50 | ~7 |
| 51–60 | ~6 |
| 61–70 | ~5 |
| 71–80 | ~4 |
| 81–100 | ~3 |
| 100+ | ~2 (prestige focus) |

> 💡 **Design Note:** Unlock density is intentionally higher in early levels to accelerate the new player experience and introduce all major systems quickly. Later levels are about depth and mastery rather than breadth.
