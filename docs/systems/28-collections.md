# 🏆 28 — Collections

Collections are themed sets of items that players gradually assemble by earning individual collection items through gameplay. Completing a set awards significant rewards and is one of the most satisfying goal structures in the game.

---

## 📦 How Collections Work

1. **Earn items** — Collection items drop from customers, harvests, quests, and achievements
2. **View progress** — The Collections panel shows progress per set
3. **Complete set** — When all items in a set are collected, claim the reward
4. **Reset** — Some collections reset after claiming (repeatable); others are one-time

## 📊 Collection Drop Rates

| Item Rarity | Drop Chance |
|------------|------------|
| Common | 25% per relevant action |
| Uncommon | 10% per relevant action |
| Rare | 3% per relevant action |
| Very Rare | 1% per relevant action |

> 💡 **Design Note:** Collection items should feel achievable within reasonable play time. Common items drop frequently enough that casual players can complete collections; rare items create memorable moments when they drop.
