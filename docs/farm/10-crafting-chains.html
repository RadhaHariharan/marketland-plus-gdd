⛓️ 10 — Crafting Chains

Multi-step crafting chains are the most profitable — and most complex — production paths in SuperStoreTycoon. They require coordinating multiple buildings, crop types, and animal products across several processing steps. The payoff is enormous: chain-end products sell for 3–10× the base value of their raw ingredients.

🔗 What Is a Crafting Chain?

A crafting chain is a production sequence where the output of one building becomes the input of another. For example:

Wheat → Bakery Oven → Bread
   ↘
   Hay → Cow → Milk → Dairy Factory → Butter → Bakery Oven → Croissants (3× value)

🏆 Key Crafting Chains

Chain 1: The Bakery Premium

Wheat (Farm) → [Bakery Oven] → Flour
Cow Milk (Farm) → [Dairy Factory] → Butter
Chicken Eggs (Farm) → Eggs
Flour + Butter + Eggs → [Bakery Oven] → Croissants (85 coins, 🌿 Fresh)

Value multiplier: 3.2× over plain bread

Chain 2: The Winemaker

Grapes (Farm, Lv.20) → [Juice Press] → Grape Juice (simple)
              ↓
Grapes + Sugarcane → [Winery] → Wine (240 coins, �� Fresh)
Grapes/Grain → [Distillery] → Whiskey (380 coins, 🌿 Fresh)

Value multiplier: 7× over raw grapes

Chain 3: The Luxury Perfume Line

Lavender (Farm) + Exotic Orchid (Greenhouse) + Vanilla (Greenhouse)
  → [Perfume Atelier] → Luxury Perfume (500+ coins, 🌿 Fresh)

Requires coordination between outdoor farm and greenhouse.
Value multiplier: 9× over lavender raw value

Chain 4: The Chocolate Empire

Sugarcane → Sugar
Vanilla (Greenhouse) → Vanilla Extract
Cow Milk → Milk
Sugar + Vanilla + Milk → [Chocolate Factory] → Artisan Chocolate (300 coins)

Value multiplier: 5× over sugarcane

Chain 5: The Cosmetics Line

Lavender → [Cosmetics Lab, Step 1] → Lavender Extract
Goat Milk → [Cosmetics Lab, Step 2] → Goat Cream
Beeswax (from Bees) → [Cosmetics Lab, Step 3] → Skincare Set (350 coins)

Value multiplier: 8× over lavender raw value

Chain 6: The Royal Feast

Truffle (Pig bonus / Greenhouse) + Saffron (Greenhouse) → [Gourmet Kitchen] → Royal Feast (800 coins)

Only available to Celebrity and Royalty/VIP customers.
Value multiplier: 10× over truffle raw value

Chain 7: The Candle & Honey Studio

Bees → Honey + Beeswax → [Honey Studio] → Luxury Honey / Artisan Candles
Lavender (feeds bees) → Higher-quality Honey (10% bonus if lavender-fed)

Value multiplier: 4× over raw honey

Chain 8: The Textile Empire

Sheep Wool → [Textile Mill] → Fabric
Cotton → [Textile Mill] → Cotton Fabric
Fabric → [Textile Mill, Step 2] → Clothing → Clothing Rack (180 coins)
Rabbit Angora Wool → [Textile Mill] → Luxury Fabric → Luxury Fashion (450 coins)

Value multiplier: 6× (standard) / 8× (luxury)

⚡ The Full Chain Achievement

Completing one full multi-step chain from raw crop to final store sale earns the 🏆 "The Full Chain" achievement (+200 XP, +50 FT).

🤖 Chain Automation (Master Crafter)

The Master Crafter worker (Lv.70) can manage entire chains autonomously:
1. Scans all buildings for available inputs
2. Prioritises chains based on player-set priority
3. Queues jobs in optimal order
4. Moves intermediate goods between buildings
5. Delivers finished goods to supply lane

💡 Design Note: Chains can feel complex, but players are never required to use them. Simple single-step products always remain profitable. Chains are a reward for invested players, not a barrier.
